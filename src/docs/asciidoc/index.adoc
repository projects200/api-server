== 개요

Undabang API 서비스에 대한 문서입니다.

[[overview-http-status-codes]]
=== HTTP 상태 코드

|===
| 상태 코드 | 설명

| `200 OK`
| 요청이 성공적으로 처리됨

| `201 Created`
| 새 리소스가 성공적으로 생성됨

| `400 Bad Request`
| 잘못된 요청 또는 파라미터

| `401 Unauthorized`
| 인증 정보가 없거나 유효하지 않음

| `403 Forbidden`
| 인증은 되었지만 권한이 없음

| `404 Not Found`
| 요청한 리소스를 찾을 수 없음

| `500 Internal Server Error`
| 서버 오류
|===

[[overview-headers]]
=== 공통 헤더

|===
| 헤더 | 설명

| `X-USER-ID`
| 사용자 식별자 (UUID 형식)

| `Content-Type: application/json`
| JSON 형식의 요청 본문

| `Accept: application/json`
| JSON 형식의 응답 본문
|===

[[overview-response-format]]
=== 응답 형식

[source,json]
----
{
  "success": "boolean",
  "code": "string",
  "message": "string",
  "data": {
    // 응답 데이터
  }
}
----
